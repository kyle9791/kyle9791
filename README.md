# 🕵️ 외국어 간판 아카이브
저긴 대체 뭐 하는 가게야…? 🤔
외국어로만 되어 있는 간판들, 옥외광고법에 어긋날 뿐더러 고령화 사회의 노인 소외 현상의 문제점으로도 꼽히는 거 알고 계셨나요?
